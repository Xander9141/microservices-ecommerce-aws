# Microservices Ecommerce AWS

Sistema ecommerce basado en arquitectura de microservicios utilizando Docker, FastAPI y AWS.

## Arquitectura

* Frontend
* Backend API REST
* Microservicio de pagos
* Docker Compose
* GitHub Actions CI/CD
* AWS ECS + ECR

## Tecnologías

* Angular
* FastAPI
* Docker
* AWS
* GitHub Actions

## Funcionalidades

* Login de usuarios
* Registro
* Carrito de compras
* Integración de pagos
* Visualización de productos
